Prework Study Guide

HTML

  • The head element contains contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The <p> element represents a paragraph or block of text.
  • The <br> element creates a line of empty space, or a line break, between two blocks of content.
  • The <li> is used to define each individual item within a list, whether it's an unordered list or an ordered list.

CSS

  • A margin indicates how much space we want outside of an element.
  • A padding indicates how much spacing we want around the content inside of the element
  • A <link> element creates the connection to the external file.
  • The rel attribute specifies the relationship between the current document and the linked document resource.
  • The href attribute specifies the location (URL) of the external resource.
  • The box-shadow CSS property adds shadow effects around an element's frame and described by X and Y offsets relative to the element, blur and spread radius, and color.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b "branch name": creates a new branch.
  • git commit -m "commit message": commits the changes along with a message of what has changed
  • git pull origin main: to verify if our local branch is in sync with the base branch in GitHub.
  • git push origin "branch name": to push the changes we have locally to our remote GitHub branch.
